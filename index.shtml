Iroqwa.org Packages

This is my Debian packages repository. Feel free to use them and drop me an email :)

To use this repository:

echo "deb http://debian.iroqwa.org/ jessie main" | sudo tee /etc/apt/sources.list.d/iroqwa.list
sudo wget http://debian.iroqwa.org/iroqwa.org.gpg.key --output-document=- | apt-key add -
sudo apt-get update
sudo apt-get install package
